##  功能实现

### 系统监控
- **硬件信息监控** - 实时监控CPU、GPU温度、频率、功耗等硬件信息
- **性能监控面板** - 直观展示系统资源使用情况

### CPU
- **短功率(SPL)控制** 
- **长功率(SPP)控制**
- **温度墙设置**
### 风扇
- **基础风扇控制** - 手动调节CPU和GPU风扇速度
- **高级自动风扇控制** - 基于温度的智能风扇调速系统
- **风扇曲线编辑** - 自定义风扇转速曲线
- **开机自启风扇系统** - 支持系统启动时自动启动风扇控制

### GPU
- **GPU直连模式** - 独显直连设置选项

### RGB 键盘控制
- **颜色自定义** - RGB颜色(R/G/B)调节
- **亮度调节** - 键盘背光亮度等级设置

### 环境光控制
- **Logo灯光管理** - Logo光开关控制

### 系统设置
- **性能模式切换**
- **开机自启** - 应用开机自启动设置
- **系统托盘集成** - 支持最小化到系统托盘
